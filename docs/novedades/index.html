    Este verano de 2021 vamos nuevamente a Guinea Bissau

    Este año vamos a realizar un nuevo viaje en verano. Vamos a ir un grupo de varias enfermeras y médicos para dar asistencia in situ, en el dispensario que tenemos en la aldea de Gambasse. También vamos a fletar un contenedor y estamos haciendo una campaña para recoger material. También queremos instalar un sistemas de placas solares en el dispensario para dotarlo de electricidad y así poder utilizar equipos médicos de electromedicina que nos permita hacer mejor los diagnósticos.
